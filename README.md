# cisco_packet_tracer_tutorials

Tutorials on using Cisco Packet Tracer.

Read the guide at [WAN Network Tutorial - Cisco Packet Tracer](./docs/index.md).

## Commands

Install all dependencies
```
pnpm install
```

Start the website
```
pnpm dev
```

Build the website
```
pnpm build
```

Preview the built website
```
pnpm preview
```
